# WellFit

Move it! Move it! is a browser-based camera game with Pose Mirror and BodyWare game modes.

## Vercel

Deploy with the Vercel root directory set to `bodygame/`.

Required environment variables:

- `UPSTASH_REDIS_REST_URL`
- `UPSTASH_REDIS_REST_TOKEN`

See `bodygame/VERCEL_DEPLOY.md` for deploy details.
